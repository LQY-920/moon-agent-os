## 30-100 万营收是关键分水岭（必读）

中国税制下，这个营收段有两条重要红线：

1. **年 500 万销售额**：小规模纳税人 ↔ 一般纳税人的自动转换阈值。你 30-100 万**远低于**这条线，默认是小规模。
2. **月 15 万（季 45 万）销售额**：小规模纳税人**免征增值税**的优惠门槛。
   - 如果你营收稳定在 30-60 万/年（月均 2.5-5 万），**全年免增值税**（0% 征收率）
   - 如果营收到 100 万（月均 8-9 万），**也还在月 15 万免征线以下**，全年仍可免增值税
   - 只有**单月突破 15 万**的那个月，才按 3% 征收（2027-12-31 前的优惠税率，原本是 3%，当前优惠 1%）

**结论**：你 30-100 万营收段在增值税上**几乎是 0 税负**。核心税种是**企业所得税** + **个人分红个税**。

## 三个必须先想清楚的核心问题

### 问题 1：一人有限公司 vs 个体工商户

很多人以为 "一人创业 = 一人有限公司"，但**个体工商户**在你这个营收段可能更划算。对比：

| 维度 | 一人有限公司 | 个体工商户 |
|---|---|---|
| **法律身份** | 法人，有限责任 | 自然人延伸，**无限责任**（公司债务 = 个人债务） |
| **企业所得税** | 25%（小微 100 万内 5%，100-300 万 5%，都是优惠） | **无**，直接按"经营所得"征个税 |
| **分红税** | 企业所得税交完后，利润分给股东再征 20% 分红个税 | 无分红概念 |
| **综合税负**（30-100 万） | 约 12-18%（企税 5% + 分红 20% 组合） | 按经营所得 5-35% 五级累进，30-100 万档位约 10-20% |
| **注册难度** | 需验资（现在认缴可 1 元）、做账、年报 | 最简单，有些城市可在税务局一次性办完 |
| **开票资质** | 一般纳税人可开专票 13%、小规模可开专票（自 2016 后放开） | 可开普票、专票（代开） |
| **能否做 SaaS 订阅收款** | ✅ 完全没问题 | ✅ 也可以，但客户心理上更愿意和"公司"签合同 |
| **未来融资** | ✅ 能融资、能股权分配 | ❌ 无法融资、无股权概念 |
| **注销难度** | 高（税务清算、工商注销、耗时 3-6 月） | 低（税务销户，个体户登记注销快） |
| **合规负担** | 月报/季报/年报 + 做账 + 审计（非强制） | 小规模核定征收可"零申报" |

**你的具体决策点**：

- **如果 6-12 个月内会考虑**融资（天使/种子），或者需要和大客户签正式合同走 B2B 流程 → **一人有限公司**
- **如果是纯面向 C 端 / 小 B 订阅、12 个月内不融资、想低成本试水** → **个体工商户**性价比高
- **关键的反事实考虑**：你 FOUNDER_CONTEXT 里写了"未来若融资方向为独立开发者友好的天使/种子"。如果融资是认真考虑的路径，个体户会卡死；一人公司虽然现在税负略高，**但它是可投资主体**

**我的倾向**：结合你"超级个体 + moon-agent-os 做平台 + 有融资想象空间"的定位 → **建议走一人有限公司**。个体户的低税负诱惑在 30-100 万量级上**一年最多省 1-2 万**，但换不来可投资的主体身份。

### 问题 2：认缴资本写多少？

认缴制下不需要实缴，但认缴额**不等于随便写**：

- 认缴额 = 你在公司负债时的**最大偿债责任**
- 认缴额越大，工商部门、客户、银行对公司的"面子判断"越高，但你兜底风险也越大

**30-100 万营收场景的合理建议**：

| 认缴额 | 适用情景 | 风险/成本 |
|---|---|---|
| **1-10 万** | 纯 SaaS 订阅、不会产生大额负债、不需要给客户"我们是正规军"印象 | ✅ 风险低，推荐起点 |
| **10-100 万** | 签对公合同会被要求"注册资本不能太低"时用（部分大客户筛选标准） | ⚠️ 万一业务失败被追债，你要补齐这个数 |
| **100 万+** | 融资前调整（投资人会要求实缴部分或调整比例） | ❌ 过度设计，前期没必要 |

**关键注意**：**2024 年新《公司法》已实施**，认缴期限**最长 5 年**（旧法无限期）。如果你认缴 50 万，5 年内必须实缴到位。写太高以后会被动"掏钱填坑"。

**推荐**：**起步认缴 10 万**，5 年内随便实缴（真拿 10 万出来放进对公账户也算实缴），够所有正常 B2B 场景用。

### 问题 3：经营范围怎么写？

大陆工商注册要写"经营范围"，这个栏位看起来是小事，但**写错了后面加业务要变更、审计要对表、开票要对应**。

**moon-agent-os 的经营范围候选**（从窄到宽）：

```
技术服务;技术开发;技术咨询;计算机软件的研究、开发、销售;
信息技术咨询服务;互联网数据服务;软件开发;人工智能软件开发;
人工智能行业应用系统集成服务;数据处理服务;
（一般项目，除依法须经批准的项目外，凭营业执照依法自主开展经营活动）
```

**关键提示**：

- **"一般项目" vs "许可项目"**：AI 软件服务走一般项目即可，不用办增值电信许可证（ICP 除外——如果做公开互联网服务、有用户登录，需要 ICP 备案，但不是经营范围问题）
- **避免写"教育培训"、"互联网信息服务"、"出版物发行"** 等许可项目，会被卡在前置审批
- **写宽不写窄**：未来 moon-agent-os 可能涉及平台服务、AI 代理、MCP 上下游，都先写进去

**"人工智能"关键词是否需要资质**：目前一般性 AI 软件开发**不需要**，但如果未来接入生成式 AI 做 2C 服务，可能触发"生成式人工智能服务管理暂行办法"的算法备案（这是另一个维度的合规，和营业执照分开）。

## 注册地决策（你选"现在所在城市"）

你选了"现在所在城市"——这个决策已经简化了。但城市内部还有细分选择：

| 注册地类型 | 说明 | 适合你吗 |
|---|---|---|
| **自家/居家办公地址** | 直接用你的公寓/租房地址。需要房东配合（开居住证明） | ✅ 最省钱，**前提**：房东不反对、小区物业不卡（很多小区明确"禁止公司注册"） |
| **创业园区 / 共享办公** | 600-3000 元/年的虚拟地址 + 基础代办 | ✅ 主流选择，无房东风险 |
| **写字楼租赁地址** | 真租办公室 | ❌ 你单人开发阶段没必要 |
| **经开区政策地址** | 异地挂靠（如本市的某个经开区有政策招商）| ⚠️ 返还政策诱人但配套复杂，30-100 万营收段吃不到大头 |

**推荐**：**创业园区 / 代理记账公司提供的虚拟地址**（当地搜"一人有限公司注册 + 代理记账"，一般打包价 1500-3000 元/年，含工商注册 + 记账 + 报税）。

## 法人代表 / 股东 / 监事配置

一人有限公司的特殊结构：

- **股东**：你（100%）
- **法定代表人**：默认你
- **监事**：**不能是你自己**，必须是另一个自然人

**关键卡点**：你需要**找一个成年人做监事**。这个人只挂名、不掌权、不承担责任，但：

- 需要身份证原件拍照
- 需要本人到场或配合电子签
- **不能是公司员工**

**推荐人选**：家人（配偶、父母、成年兄弟姐妹）或关系密切的朋友。**提前沟通好，避免临办理时卡住**。

## 对公账户开户（最耗时的一步）

营业执照拿到后，下一步是开对公基本户。这是你需要**真正亲自跑**的一步：

1. **选银行**：
   - **建设银行 / 工商银行 / 中国银行**：服务网点多、客户对公打款熟悉，但审核严、网点挑剔
   - **招商银行**：年轻创业者友好，App 体验最好，开户流程相对简化
   - **平安银行 / 民生银行**：小微客户友好
   - **微众银行（线上）**：全程线上开户，但部分企业客户可能不接受从微众付款

2. **必须本人到场面签**（中国对公开户反洗钱要求，不可代办）
3. **带齐资料**：营业执照正副本、公章/财务章/法人章、身份证、注册地租赁合同或产权证明
4. **耗时**：1-4 周（取决于分行审核队列）

**关键提示**：开户后有**U 盾 / 数字证书**，丢了或坏了是噩梦。**立即备份、妥善保管**。

## 3 个常见坑（线下办事前必读）

### 坑 1：核名撞车

"AI"、"智能"、"科技"、"月球"、"Moon"、"Agent" 这种热词撞名概率极高。**准备 3-5 个候选字号**，让代办一次性核完，别一个个试。

### 坑 2：监事临时找不到

监事必须是另一个自然人、**不能是股东本人**。很多人办到这一步才发现找不到合适的人。**提前沟通**，避免卡两周。

### 坑 3：注册地址被"小区禁止注册"卡住

很多小区住宅明确不允许商事注册。**如果选自家地址，提前让物业/房东出具允许注册的证明**（很多物业不配合）。这就是为什么我推荐**虚拟地址**——一次花 1500-3000 元买断烦恼。

## 我的整体推荐（基于你三个关键输入的组合判断）

**推荐路径**：

1. **形态**：**一人有限公司**（为未来可能的融资留空间，放弃个体户的微量税负优势）
2. **注册地**：**你所在城市的虚拟地址 + 代理记账一体服务包**（1500-3000 元/年）
3. **认缴资本**：**10 万**
4. **经营范围**：写宽版（技术开发 + 技术服务 + AI 软件 + 数据处理一口气都写上）
5. **监事**：配偶 / 父母 / 密友，提前 1 周沟通
6. **开户银行**：**招商银行** 首选，**工商/建行** 备选
7. **时间**：**等 MVP M2 跑通后再办** —— 理由：
   - 你现在开公司等于每月烧 200-500 元记账费 + 报税工时，**0 收入不建议**
   - MVP 跑通后会产生第一笔收入迹象（访谈反馈、早期付费意向），那时开户也更有动力
   - 新公司法认缴期 5 年，不用着急
   - **"充分情况下立刻开干"的"充分"在你这里应是：MVP 跑通 + 至少 1 个口头付费意向**
